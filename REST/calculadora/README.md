# Questão 1: Cliente REST - Calculadora Remota

Este projeto consiste na implementação de um cliente REST desenvolvido em Python para interagir com a API de Calculadora especificada na atividade.

O cliente é capaz de consultar as operações disponíveis e executar cálculos (soma, subtração, divisão e multiplicação) enviando requisições para o servidor remoto hospedado no Render (https://calculadora-fxpc.onrender.com).

## Como Executar

### Pré-requisitos
1.  **Python 3** instalado.
2.  **Biblioteca `requests`** instalada:
    ```bash
    pip install requests
    ```
    
### Passo 1: Executar o Cliente
    ```bash
    python cliente_calculadora.py
    ```
